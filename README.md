# PL0 Compiler

note: the SLR syntax parser in `LR_table_gen` is from [Vipul197/slr-parser](https://github.com/Vipul97/slr-parser)

environment: Linux

requirement: gcc

## usage

### just build

```
make all
```

### build and run

```
make run
```

### run

```
./main prog.pas
```

---

note:

- `run` means compile and run the program.

- it is not the same with original PL/0 assembly code.
